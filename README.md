# Habit-Tracker

## 프로젝트 소개 💡

```
취미 생활을 기록하는 페이지로 새로운 취미 리시트를 등록할 수 있고, 등록된 리스트를 카운트 할 수 있다.
리스트의 총 횟수를 헤더 타이틀에서 확인 할 수 있다.
기록들을 모두 리셋할 수도 있고, 리스트들을 각각 제거할 수 있다.
새로운 취미를 등록하는 것은 input form을 통해서 추가해 취미 생활을 기록하는 Habit-Tracker이다.
```
